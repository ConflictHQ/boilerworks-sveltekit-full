# Boilerworks SvelteKit Full

> Edge template -- Full-stack SvelteKit app on Cloudflare Pages. SSR + client + API routes in one framework.

**Status:** Planned

SvelteKit full-stack edge application on Cloudflare Pages. Svelte 5 with runes, server-side rendering, client hydration, and API routes all in one framework. No separate backend needed. Ideal for dashboards, SaaS apps, internal tools, and any full-stack app that benefits from edge deployment.

**Edge template** -- production deployment targets Cloudflare Pages, not Docker.
